# 🎮 Rock Paper Scissors Game (Python)

A simple command-line **Rock Paper Scissors** game developed using **Python**. This project allows the user to play against the computer, which makes random choices using Python's built-in `random` module.

This project was created to practice Python fundamentals such as user input, dictionaries, conditional statements, and random number generation.

 📌 Features

- 🎲 Play against the computer
- 🤖 Random computer-generated moves
- 🎯 Displays both player and computer choices
- 🏆 Announces the winner or a draw
- 💻 Beginner-friendly console application

## 🚀 Getting Started

### Prerequisites

- Python 3 installed on your computer

### Installation

Clone the repository:

```bash
git clone https://github.com/aarohi3612/rock-paper-scissors.git
```

Navigate to the project folder:

```bash
cd rock-paper-scissors
```

Run the program:

```bash
python main.py
```

---

## 🎮 How to Play

Enter one of the following choices when prompted:

| Input | Choice |
|-------|--------|
| `r` | Rock |
| `p` | Paper |
| `s` | Scissors |

The computer will randomly select its move, and the winner will be displayed.
